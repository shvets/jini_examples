Common Tests

To try this example, run the following commands:

>start ant start.services
>ant tests

First command will launch all required Jini services, like reggie, service browser etc. Second command executes TestNG unit tests:
  • Lookup Service test;
  • Service List test;
  • Lookup Discovery test;
  • Lookup Discovery Manager test;


    • You will see TestNG output and results summary. If everything is OK, you can go to next example.
