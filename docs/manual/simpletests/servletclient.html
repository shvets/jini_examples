Client as Servlet/JSP

Now let's try to start the client for our simple service from the servlet. I created servlet (in form of JSP page) for wrapping this client (JiniTest,jsp).

The web server that supports JSPs/servlets should be started. I use tomcat for this purpose.

You have to specify the location of Tomcat Application Server. Do it right now by modifying tomcat.home property in the project.properties file.

Start tomcat web server:

>start ant tomcat.start

Finally, start your favorite browser and write down the following request:

>http://localhost:8080/JiniTest.jsp

Be sure that this service is already registered (or still registered). If not, do the following:

>ant sservice.service

Also, don't forget to run standard Jini services:

>ant start.services

Now you are ready for the next level.
