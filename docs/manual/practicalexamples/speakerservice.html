Speaker Service

In this example we'll make small experiment with FreeTTS library. It is a speech synthesis system written entirely in the Java language.

After the build we have 2 archives:
  • for the service itself (lib-service/speaker-service.jar);
  • for the client that wants to work with this service (docroot/service/speaker-service-dl.jar).


    • You can test this example now:

    >start ant start.services
    >start ant speaker.service
    >ant speaker.client

    Client should list names of all available voices and the it pronounce the phrase: "Hello, Jini World".
