Blitz JavaSpace test

Blitz is separate implementation of JavaSpace. So, there is no special instructions how to work with it. You only need to run this implementation instead of "outrigger". Use this command:

>start ant blitz.services

instead of "start.services" command.

Configuration is located here: src\main\config\start-blitz-all.config and for blitz itself: src\main\config\blitz.config.

Dependent services list:
- reggie;
- blitz;
- http server for Jini libraries;
- http server for this service;

Now you can run any of these commands:

>ant blitz.stats
>ant blitz.dashboard
>ant blitz.entrysizer
